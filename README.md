### Description

My personal site

#### Install 


	bundle update
	bundle install
	npm install
	sudo npm install -g grunt grunt-cli

#### Build
Run

	grunt watch
	bundle exec jekyll serve --watch --host 0.0.0.0

Generates to `_site`.

`grunt-jekyll` no longer works because the latest jekyll is not supported by this grunt plugin.
